# Mac menu bar (lands at M2)

v0 is a SwiftBar plugin: today's agenda, recent notes, and a "New note…"
dialog that POSTs to the forge-server API. v2 replaces it with a Tauri tray
app: global hotkey (⌥Space) popping the web app's `/capture` route in a small
always-on-top window, sharing the exact same UI and tokens as every other
surface.
